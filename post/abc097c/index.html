制約(Kは1以上5以下となる。)から, 生成できる部分文字列のうち, 上位5番目までの 文字列の長さはK以下となる。 １文字目から順に文字列長K以下となる部分文字列を生成・記録し, K番目の文字列を出力すれば良い。

以下提出コードです。

use std::collections::HashSet;

fn read<T: std::str::FromStr>() -> T {
    let mut s = String::new();
    std::io::stdin().read_line(&mut s).ok();
    s.trim().parse().ok().unwrap()
}

fn main() {
    let v: String = read::<String>();
    let k: usize = read::<usize>();
    let mut hs: HashSet<String> = HashSet::new();

    for i in 0..v.len() {
        for j in 0..k {
            if i+j < v.len() {
                hs.insert(v[i..=i+j].to_string());
            }
        }
    }

    let mut vv: Vec<String> = hs.into_iter().collect();
    vv.sort();
    println!("{}", vv[k-1]);
}
