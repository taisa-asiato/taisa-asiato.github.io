はじめはa1からb3までの和がc11からc33までの和の1/3に等しいので すべての入力を和算したのち3で割り切れるかどうかで判定したがWAとなった。

解説のとおり, a1を0と仮定しても差し支えないため, そのような実装にしました。

以下提出コードです。

fn read<T: std::str::FromStr>() -> T {
    let mut s = String::new();
    std::io::stdin().read_line(&mut s).ok();
    s.trim().parse().ok().unwrap()
}

fn read_vec<T: std::str::FromStr>() -> Vec<T> {
    read::<String>().split_whitespace()
        .map(|e| e.parse().ok().unwrap()).collect()
}

fn main() {
    let mut v: Vec<Vec<i32>> = Vec::new();
    let mut a: [i32; 3] = [0; 3];
    let mut b: [i32; 3] = [0; 3];
    for i in 0..3 {
        let tmp: Vec<i32> = read_vec::<i32>();
        v.push(tmp);
    }

    b[0] = v[0][0] - a[0];
    b[1] = v[0][1] - a[0];
    b[2] = v[0][2] - a[0];

    a[1] = v[1][0] - b[0];
    a[2] = v[2][0] - b[0];

    if a[1]+b[1] == v[1][1] && a[1]+b[2] == v[1][2] &&
        a[2]+b[1] == v[2][1] && a[2]+b[2] == v[2][2] {
            println!("Yes");
    } else {
        println!("No");
    }
}
